# Искра: информация для партнёров

Искра — рабочая AI-платформа 2030AI для команд, которым нужен управляемый контур использования AI: чат, агентские сценарии, работа с файлами, база знаний, пространства, организации, интеграции и варианты развёртывания.

Этот репозиторий помогает партнёрам быстро понять, что можно продавать, как объяснять возможности продукта и какие ограничения важно проговаривать честно.

## Для кого

- партнёры и интеграторы 2030AI;
- команды продаж и pre-sale;
- руководители внедрения;
- специалисты, которые готовят клиентские материалы по Искре.

## Как пользоваться

Начинайте с обзора и индекса возможностей. Перед отправкой клиенту формулировок про безопасность, данные, специальные контуры внедрения или LLM-условия сверяйтесь с разделом legal claims.

## Что важно

Репозиторий содержит публичный curated layer. Он не заменяет договор, спецификацию, КП, техническое обследование или индивидуальное согласование контура внедрения.
