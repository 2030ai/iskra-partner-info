# Project Instructions

## Project

This repository is the public partner information base for Iskra by 2030AI.

## Purpose

- Give partners current public information about Iskra capabilities, commercial options, deployment formats, and safe sales wording.
- Keep public content in Markdown so humans and agents can review changes through Git.
- Avoid publishing internal CRM data, credentials, customer information, private deal context, or unconfirmed legal/compliance claims.

## Content Boundaries

- Public content lives in `README.md`, `CHANGELOG.md`, `LICENSE.md`, and `docs/`.
- Agent maintenance content lives in `agent_docs/`.
- Product information is curated from CRM and `2030ai-platform`; this repository is not the source for code behavior.
- Do not add JSON/YAML partner-facing data feeds.
- GitHub Actions YAML is allowed only for CI configuration.

## Safety

- Never claim confirmed Russian software registry status without a registry number.
- Never claim FSTEK certification, full 152-FZ compliance, full local processing, or no external LLM transfer without an exact deployment contour and reviewed wording.
- Use relative links for internal navigation.
- Do not commit local absolute home paths or session paths.
